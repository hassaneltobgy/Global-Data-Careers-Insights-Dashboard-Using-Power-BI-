# Global-Data-Careers-Insights-Dashboard-Using-Power-BI-
interactive Power BI dashboard based on a survey of over 600 data professionals, focusing on extracting actionable insights related to salary trends, job satisfaction, and skill demand across different roles

performed data cleaning and transformation using Power Query, including standardizing categorical fields such as job titles and countries, and transforming salary ranges into structured formats suitable for analysis along with other data cleaning steps like removing unwanted columns


# Dataset Analysis :
Data scientist and Data Engineer have highest salaries 
Python emerged as the dominant programming language across all roles, highlighting its importance in the data ecosystem
Despite moderate work-life balance satisfaction, salary satisfaction was noticeably lower, indicating a gap between expectations and compensation.
I focused not only on visualization but also on making the dashboard insight-driven and decision-oriented.
